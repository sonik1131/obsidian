# Obsidian Vault — Научные конференции

Хранилище конференций по тематикам **IT** и **АПК** (агропромышленный комплекс) с индексацией в **РИНЦ/ВАК**.

## Структура

- **`Conferences.md`** — главная таблица всех конференций (сводка)
- **`Conferences/`** — отдельные заметки на каждую конференцию
- **`Articles/`** — наши статьи и публикации
- **`Authors/`** — информация об авторах-соавторах
- **`Templates/`** — шаблоны для новых записей

## Статус

🗓 Автообновляется через cron (коммиты с сервера)
🔄 Синхронизация с MacBook через Obsidian Git
